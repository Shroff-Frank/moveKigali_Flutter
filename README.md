# MoveKigali Flutter Project Structure

## lib/
- `main.dart` → app entry point
- `screens/` → all screens/pages
- `widgets/` → reusable widgets
- `utils/` → constants, colors, helpers

## assets/
- `images/` → app logos, backgrounds
- `icons/` → icons used in UI

## Tips
- Keep all assets in `assets/` folder
- Name files clearly: `logo.png`, `bus_icon.png`
- Import using `Image.asset('assets/images/logo.png')`
